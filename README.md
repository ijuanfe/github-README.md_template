# Artificial Intelligence: Project 1

## To run the script:

```
$ sh run.sh -s <search algorithm: 1, 2, or 3> -f <path to: levelfile.txt>
```

## Where:

|Option|Search Algorithm|
|:-----------|:------------------------------------------------------------|
|`1`|Depth-first search|
|`2`|Breadth-first search|
|`3`|Iterative deepening depth-first search|
